Recent Posts Widget with truncated post titles.

== Description ==
Maintain layout formatting when post titles are too long. Plugin allows admin to set maximum title characters and define truncation padding.

== Installation ==
1. Upload `pm-recent-posts.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Add widget to sidebar and define options
